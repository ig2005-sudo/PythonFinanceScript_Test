# PythonFinanceScript_Test
# Project Overview
This repository contains a technical pipeline designed to bridge operational business insights with quantitative financial analysis. The project automates the extraction of market data to calculate key risk-adjusted performance metrics.

# Features
- Basic Automated Data Retrieval: Real-time data extraction via Yahoo Finance API.
- Quantitative Analysis: Implementation of the Sharpe Ratio to evaluate risk/return profiles.
- Data Processing: Large-scale data manipulation using Python (Pandas).
- Visualization: Dynamic financial dashboards built in risk/reward map, stock performance over 1 year and portfolio report


Tech Stack & Workflow
This project utilizes AI for its integrated analysis of the stock curves.

Language: Python 3.14 (Pandas, NumPy, matplotlib, yfinance)
